# React-Typescript-Calendar

### Dev environment:

Set correct configuration in .env file.

> npm run start-dev

### To launch on windows:

Git clone or download project.
Set correct configuration in .env file.

> npm install

> npm run build-w

> npm run start-w

### To launch on centOS:

Git clone or download project.
Set correct configuration in .env file.

> npm install

> npm run build

> sudo npm run start
